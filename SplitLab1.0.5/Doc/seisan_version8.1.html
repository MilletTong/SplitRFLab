Latest changes

GENERAL

 Several minor bugs have been fixed

 SEED

Several smaller problems have been fixed. A major problem was that writing on PC was not correct, however since the reading routines were made to read the wrong output, we did not notice. The routines have now been fixed.

The unofficial standard is to write everything in the Sun way meaning big endian. This is now the default for SEISAN on all platforms. However, SEISAN can also be set up to write little endian on all platforms if desired, see routine seed.for in LIB.

NOTE: Files written in miniseed on PC on version 8.1 might not be readable by others.

There are still some Steim2 files SEISAN cannot read.

 

SEISAN2MSEED

Official IRIS  program to convert  from SEISAN to mseed, all platforms and all mseed formats.

Wavetool

Outputs have been changed from extract.out to wavetool.out and a few bugs fixed.

Output option –wav_out_file=SEISAN means that a standard SEISAN file name is made instead of wavetool.out.

Output option –duration is in seconds.

Mb magnitude.

Magnitude is NOT calculated if the phase is blank

Magnitude is only calculated if distance is larger than test(57)

Ml and Mc

If both were calculated, Mc came before Mc on header line, now it is the opposite. This is to reflect that Ml is the more important magnitude.

ISC location program: Does not work on PC

 

Phase weight key (manual p104). To clarify: The symbol used for weight must be in column 41 and the weight in 51. Weight is a one digit integer.
